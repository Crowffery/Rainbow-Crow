# Privacy Policy

This repository hosts a public, static privacy policy page for Google Play Console compliance. It is intended to be published through GitHub Pages at `https://<github-username>.github.io/Rainbow-Crow/` and linked from the Google Play Console Store Listing.

直接访问链接（已替换为本仓库路径）：https://crowffery.github.io/Rainbow-Crow/

## 常见 404 原因
- GitHub Pages 的项目路径包含仓库名，如果仅访问 `https://<github-username>.github.io/` 而没有加上 `/Rainbow-Crow/`，会出现 404。
- App Store 或广告平台配置的隐私政策链接如果遗漏仓库名或拼写有误，同样会跳转到 404。
- 旧缓存可能仍指向错误链接，可以清除缓存或直接访问上方的完整链接验证。

## 404 页面
仓库中提供了 `404.html`，会引导用户返回正确的隐私政策主页并提示使用完整的 GitHub Pages 地址。

The policy presents disclosures for Android apps distributed through Google Play by the individual developer Jeffery. The site uses only HTML and CSS, with no tracking scripts or backend services.
