# Privacy Policy

This repository hosts a public, static privacy policy page for Google Play Console compliance. It is intended to be published through GitHub Pages at `https://<github-username>.github.io/privacy-policy/` and linked from the Google Play Console Store Listing.

直接访问链接（已替换为本仓库路径）：https://crowffery.github.io/Rainbow-Crow/

The policy presents disclosures for Android apps distributed through Google Play by the individual developer Jeffery. The site uses only HTML and CSS, with no tracking scripts or backend services.
