# Infernal Gambit (深渊赌局)

Welcome to the abyss. This is the official landing page for **Infernal Gambit**, a hardcore strategy Roguelike game where the abyss is the board and souls are the currency.

## Live Site
Check it out here: [https://crowffery.github.io/Rainbow-Crow/](https://crowffery.github.io/Rainbow-Crow/)

## Project Structure
- `index.html`: Main landing page for Infernal Gambit.
- `style.css`: High-end "Abyssal Cyber" styling for the main site.
- `privacy.html`: Privacy Policy for Google Play compliance.
- `privacy.css`: Styling for the privacy policy page.
- `404.html`: Custom themed error page.

## Deployment Note
This site is hosted via **GitHub Pages**. 
If you see a 404 error, please ensure:
1. The changes are pushed to the `main` branch.
2. GitHub Pages is enabled in `Settings > Pages` (Source: Deploy from a branch -> `main`).
